# Home Assistant Add-on: Tinyauth

![Tinyauth Version](https://img.shields.io/badge/Tinyauth-v4.1.0-blue)

> ⚠️ **WARNING: This add-on is not fully tested. Please be careful and use at your own risk.**

Tinyauth is a lightweight, powerful authentication proxy designed to provide a unified login experience. It supports:
- **OAuth2/OIDC**: Google, GitHub, Microsoft, and custom providers.
- **LDAP**: Integration with Active Directory or OpenLDAP.
- **Local Users**: Simple user/password authentication.
- **Responsive UI**: Modern, clean, and customizable login interface.

## Installation

You can add this repository to your Home Assistant instance by clicking the button below:

[![Add repository on my Home Assistant][repository-badge]][repository-url]

1. Click the button above.
2. Search for "Tinyauth" in the Add-on Store.
3. Click **Install**.
